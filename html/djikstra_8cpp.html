Detailed Description

Computes the shortest paths from a starting vertex to all other vertices in the graph using Dijkstra's algorithm.

This function uses a priority queue to efficiently find the shortest path from the starting vertex to all other vertices. The graph is represented by an adjacency matrix, where adjMatrix[u][v] holds the weight of the edge from vertex u to vertex v.

Parameters
startThe starting vertex for Dijkstra's algorithm.
Returns
A vector of integers where the value at each index represents the shortest distance from the starting vertex to that vertex.
