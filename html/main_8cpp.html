Detailed Description

Entry point for the Chinese Postman problem solver.

Version
1.0 author Jakub Doman Compilation Make compile

This program reads a graph from a JSON file, initializes the graph, and solves the Chinese Postman problem using different algorithms.

Parameters
argcNumber of command-line arguments.
argvArray of command-line arguments.
  • argv[0]: Program name
  • argv[1]: Path to the JSON file containing the graph
  • argv[2]: Number of postmen
  • argv[3]: Seed for random number generation
Returns
int Exit status of the program.
  • 0: Success
  • 1: Error

Usage:

./main <json file> <number of postmen> <seed>
